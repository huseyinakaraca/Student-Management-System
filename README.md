# Student Management System

A personal side project I developed to practice C# Windows Forms and MS SQL Server. I didn't write this for a class assignment; I just wanted to build something from scratch to understand how layered architecture works.

**Key Features:**

* Basic CRUD (Create/Read/Update/Delete) operations connected to an SQL database.
* Live search feature.
* Simple input validations (prevents saving duplicate student numbers).
* Separation of UI and database logic.
